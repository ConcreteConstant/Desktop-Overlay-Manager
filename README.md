# Desktop Overlay Manager

A Windows desktop application that spawns image, audio, and video overlays on your screen at configurable intervals.

Designed for ambient visuals and background media.

## Usage
1. Launch the application
2. Add one or more media folders
3. Adjust spawn behavior and overlay settings
4. Let it run in the background

## Features
- Image, audio, and video overlays
- Random and fullscreen presentation types
- Click-through or interactive overlays
- Per-media spawn chance, lifetime, scale, and weight controls
- Multi-monitor support
- Persistent configuration
- Control panel UI for live tuning
